# Detective Game App

#### Aplicación móvil interactiva basada en un sistema de investigación. El usuario asume el rol de detective para resolver un caso principal mediante la gestión de pistas en un tablón, interrogatorios dinámicos con tres tipos de diálogos (Pasivo, Neutral, Agresivo) y exploración de escenas del crimen.



#### Tecnologías Utilizadas 

* ##### Lenguaje: KotlinUI: Jetpack Compose (Declarativa)
* ##### Arquitectura: MVVM con Principios SOLID y POO. 
* ##### Navegación: Jetpack Navigation Component.
* ##### Gestión de Estado: ViewModel y LiveData/StateFlow.



#### Instrucciones para ejecutar la app

1. ##### Clonar el repositorio: git clone https://github.com/Janordonez/Obsidian.
2. ##### Abrir el proyecto en Android Studio Jellyfish o superior.
3. ##### Sincronizar el proyecto con los archivos de Gradle.
4. ##### Seleccionar un emulador con API 24 o superior.
5. ##### Presionar el botón Run (flecha verde) para desplegar en el dispositivo.



#### Flujo de la Aplicación

##### Inicio: Presentación del caso principal.

##### Tablón: Gestión y asignación de las 10 pistas a los 4 sospechosos.

##### Interrogatorios: Selección de diálogos para obtener información.

##### Lugares: Resolución de acertijos para desbloquear evidencias.
